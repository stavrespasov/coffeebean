This is a game created by COFFE BEAN for the Human-Computer Interaction for Mobile Application Design course (University of Primorska - Ben Gurion University).

The purpose of this game is to show rapid and involuntary change of behavior sometimes associated with a mental disorder or trauma.

Please rotate your smartphone in landscape mode and make sure your sound is on for the best experience while playig our game.

Before You play the game we recommend You to check out the characters first, You can find that by clicking on the button named "CHARACTERS".

Start the application by opening the index.html file!
